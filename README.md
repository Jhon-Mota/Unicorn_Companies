## O que foi feito no projeto
#Ferramentar utilizadas: Excel, SQL e Power BI.

- **Uso do Excel para a divisão de dados e colunas** por delimitador (`,`);
- **Formatação das datas**;
- **Formatação da coluna _Valuation_** para o tipo contábil no formato **Dólar**;
- **Importação de dados** no **SQL Server**;
- **Consulta de dados** no banco de dados usando **SQL**;
- **Criação de dashboards** para visualização de informações.
